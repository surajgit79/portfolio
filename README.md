# My Portfolio

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design** - Works seamlessly on all devices
- **Modern UI** - Clean and professional interface with smooth animations
- **Sections Included**:
  - Hero section with call-to-action
  - About me
  - Projects showcase
  - Skills display
  - Contact section

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Customize the content with your information

## Customization

Edit the following to personalize:
- Replace "My Portfolio" with your name
- Update project information and links
- Add your social media links in the contact section
- Modify colors in `styles.css` (look for `:root` variables)

## Built With

- HTML5
- CSS3 (with Grid and Flexbox)
- Vanilla JavaScript
